## Документация проекта Fleur de Rêve

### Эта папка содержит вспомогательную документацию интернет‑магазина цветов **Fleur de Rêve**.

---

### Структура документации

#### Основные файлы

- **README.md** (в корне репозитория) — главное описание проекта, установка, запуск, обзор функционала.
- **docs/README.md** — описание структуры документации (этот файл).
- **diagrams.png** — визуальная ER‑диаграмма базы данных (модели Users, Products, Cart, Orders, Reviews, CustomBouquetRequest и т.д.).
- **architectures.png** — архитектурная схема приложения (Browser → Django Backend → Database).


---

### Папка `screenshots/`

Содержит скриншоты основных экранов приложения:

- **home.png** — главная страница.
- **about.png** — страница о нас.
- **collection.png** — каталог товаров с фильтрами.
- **popular.png** — каталог бестселлеров и статистика.
- **reviews.png** — страница отзывов.
- **custom_bouquet.png** — форма заявки на авторский букет.
- **login.png / register.png** — страницы входа и регистрации.

---

### Обновление документации

При изменениях в проекте рекомендуется обновлять:

- **README.md** — при изменении функционала, процесса установки или структуры проекта.
- **diagrams.png** — при изменении моделей данных (добавление/удаление полей или таблиц).
- **architectures.png** — при изменении общей архитектуры (новые сервисы, API и т.п.).
- **screenshots/** — при заметных изменениях пользовательского интерфейса.
